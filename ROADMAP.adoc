**Phase:** Early Development (v0.1.0-alpha)

**Working:**
- ✅ ReScript frontend structure (rescript-tea + cadre-tea-router + a2ml)
- ✅ WokeLang SSG engine skeleton (ssg/main.woke)

== Future Directions

=== Post-1.0 Features
* **Plugin System** - Extend SSG with WokeLang plugins
* **Live Preview** - Dev server with hot reload
* **Incremental Builds** - Only rebuild changed pages
* **Image Optimization** - Automatic image processing
* **Search Integration** - Built-in search with consent
* **i18n Support** - Multi-language sites
* **RSS/Sitemap** - Auto-generation

=== Integration Goals
* **poly-ssg-mcp** - MCP server for SSG operations
* **wokelang.org** - Official WokeLang website
* **playground.wokelang.org** - Interactive WokeLang playground
* **Hyperpolymath sites** - Use for project documentation
